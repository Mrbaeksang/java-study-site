🔢 Wrapper 클래스 API 정리

🛠️ Integer.parseInt()

  • 풀네임: Integer.parseInt
  • 한글 뜻: 문자열을 int로 변환
  • 속한 클래스: Integer
  • 중요도: ⭐⭐⭐⭐⭐ (매우 자주)
  • 사용빈도: 매우 자주
  • 주요 사용처: 입력값 숫자 변환
  • 암기 키워드: parse = 해석, 변환
String s = "123";
int num = Integer.parseInt(s);
System.out.println(num); // 123

🛠️ Double.parseDouble()

  • 풀네임: Double.parseDouble
  • 한글 뜻: 문자열을 double로 변환
  • 속한 클래스: Double
  • 중요도: ⭐⭐⭐⭐ (자주)
  • 사용빈도: 자주
  • 주요 사용처: 입력값 실수 변환
  • 암기 키워드: parse = 해석, 변환
String s = "3.14";
double num = Double.parseDouble(s);
System.out.println(num); // 3.14

🛠️ String.valueOf()

  • 풀네임: String.valueOf
  • 한글 뜻: 다른 타입을 문자열로 변환
  • 속한 클래스: String
  • 중요도: ⭐⭐⭐⭐⭐ (매우 자주)
  • 사용빈도: 매우 자주
  • 주요 사용처: 숫자 → 문자열 변환
  • 암기 키워드: valueOf = ~의 값
int num = 100;
String s = String.valueOf(num);
System.out.println(s); // "100"

🛠️ Boolean.parseBoolean()

  • 풀네임: Boolean.parseBoolean
  • 한글 뜻: 문자열을 boolean으로 변환
  • 속한 클래스: Boolean
  • 중요도: ⭐⭐⭐ (가끔)
  • 사용빈도: 가끔
  • 주요 사용처: 입력값 true/false 변환
  • 암기 키워드: parse = 해석, 변환
String s = "true";
boolean b = Boolean.parseBoolean(s);
System.out.println(b); // true
