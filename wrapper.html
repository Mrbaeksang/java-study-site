📚 Wrapper 클래스 개요

Wrapper 클래스는 기본 데이터 타입(primitive type)을 객체로 감싸는 클래스입니다.

주요 특징

  • 기본 타입을 객체로 변환
  • 컬렉션에서 사용 가능
  • null 값 처리 가능
  • 유용한 메서드 제공
기본 타입 Wrapper 클래스 크기
byte Byte 8비트
short Short 16비트
int Integer 32비트
long Long 64비트
float Float 32비트
double Double 64비트
char Character 16비트
boolean Boolean 1비트

🎁 박싱과 언박싱

박싱 (Boxing)

기본
자동 변환

기본 타입을 래퍼 클래스의 객체로 변환하는 과정

// 수동 박싱 Integer num = new Integer(10);
// 자동 박싱 (권장) Integer num = 10;

언박싱 (Unboxing)

기본
자동 변환

래퍼 클래스의 객체를 기본 타입으로 변환하는 과정

Integer num = 10;
// 수동 언박싱 int value = num.intValue();
// 자동 언박싱 (권장) int value = num;

성능 고려사항

불필요한 박싱/언박싱은 성능 저하의 원인이 될 수 있습니다. 기본 타입을 사용할 수 있는 상황에서는 기본 타입을 사용하세요.

🛠 유용한 메서드

parseInt()

변환
자주 사용
Integer.parseInt(String s)

문자열을 정수로 변환합니다.

String str = "123";
int num = Integer.parseInt(str); // 123

valueOf()

변환
객체 생성
Integer.valueOf(String s)

문자열이나 기본 타입을 래퍼 객체로 변환합니다.

Integer num = Integer.valueOf("123"); // Integer 객체
Double d = Double.valueOf("3.14"); // Double 객체

toString()

변환
문자열
toString()

래퍼 객체를 문자열로 변환합니다.

Integer num = 123;
String str = num.toString(); // "123"

💡 실전 예제

1. 문자열 숫자 처리

public static int sumStringNumbers(String... numbers) { int sum = 0; for (String number : numbers) { sum += Integer.parseInt(number.trim()); } return sum; } // 사용 예 int result = sumStringNumbers("10", "20", "30"); // 60

2. null 안전 비교

public static boolean compareIntegers(Integer a, Integer b) { if (a == null || b == null) { return false; } return a.equals(b); } // 사용 예 boolean result = compareIntegers(Integer.valueOf(10), Integer.valueOf(10)); // true

3. 진수 변환

// 10진수를 2진수 문자열로 String binary = Integer.toBinaryString(10); // "1010" // 16진수로 변환 String hex = Integer.toHexString(255); // "ff" // 8진수로 변환 String octal = Integer.toOctalString(10); // "12"

⚡ 성능 최적화 팁

1. 캐시 활용

자주 사용되는 값은 캐시를 활용합니다.

// 권장 (-128 ~ 127 사이의 값은 캐시됨) Integer num1 = Integer.valueOf(100);
// 비권장 (항상 새 객체 생성) Integer num2 = new Integer(100);

2. 적절한 메서드 선택

상황에 맞는 메서드를 선택하세요.

// 객체가 필요한 경우 Integer num = Integer.valueOf("123");
// 기본 타입이 필요한 경우 int value = Integer.parseInt("123");

주의사항

  • == 연산자 대신 equals() 메서드 사용
  • 불필요한 객체 생성 피하기
  • null 체크 필수
