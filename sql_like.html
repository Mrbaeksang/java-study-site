🔍 LIKE 연산자

문자열 패턴을 사용하여 데이터를 검색하는 방법을 배워보세요.

1. LIKE 기본 구문

LIKE 연산자는 문자열 패턴 매칭을 위해 사용됩니다.

📝 예제: 이름이 '김'으로 시작하는 학생 조회

SELECT * FROM students
WHERE name LIKE '김%';

2. 패턴 매칭 기호

기호 설명 예시
% 0개 이상의 임의의 문자 김%
_ 정확히 한 글자 김_
[] 대괄호 안의 한 글자 김[A-Z]

📝 예제: 이름이 두 글자인 학생 조회

SELECT * FROM students
WHERE name LIKE '__';

📝 예제: 이름이 '김'으로 시작하고 두 글자인 학생 조회

SELECT * FROM students
WHERE name LIKE '김_';

💡 팁: LIKE 성능 최적화

LIKE 연산자는 성능에 영향을 줄 수 있습니다. 가능하면 정확한 검색을 사용하는 것이 좋습니다.

3. ESCAPE 키워드

특수 문자를 그대로 사용하려면 ESCAPE 키워드를 사용합니다.

📝 예제: 이름에 '%'가 포함된 학생 조회

SELECT * FROM students
WHERE name LIKE '%\%%' ESCAPE '\\';
