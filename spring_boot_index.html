🚀 스프링 부트(Spring Boot) 완전 정복

스프링 부트를 체계적으로 배우기 위한 종합 가이드입니다. 초보자부터 고급 사용자까지 단계별로 학습할 수 있습니다.

🔰 스프링 부트 기초

1️⃣ 시작하기

  • 스프링 부트 소개 초급
  • 개발 환경 설정
  • 첫 번째 애플리케이션
  • 핵심 개념 이해하기

2️⃣ 웹 개발 기초

  • RESTful API 개발 초급
  • MVC 패턴 이해
  • 정적 리소스 처리
  • 템플릿 엔진 (Thymeleaf)

💾 데이터 액세스

3️⃣ 데이터베이스 연동

  • Spring Data JPA 중급
  • JDBC Template
  • Querydsl
  • 트랜잭션 관리

🔒 보안

4️⃣ 스프링 시큐리티

  • 기본 인증/인가 중급
  • OAuth2 연동
  • JWT 인증
  • 보안 테스트

⚡ 고급 기능

5️⃣ 성능 최적화

  • 캐싱 전략 고급
  • 비동기 처리
  • 모니터링 (Actuator)
  • 성능 프로파일링

6️⃣ 마이크로서비스

  • 마이크로서비스 아키텍처 고급
  • 서비스 디스커버리 (Eureka)
  • API 게이트웨이 (Gateway)
  • 중앙 집중식 구성 관리
