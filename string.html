📚 Java API 통합 암기카드 - String

🛠️ length()

  • 풀네임: length
  • 한글 뜻: 문자열 길이 반환
  • 속한 클래스: String
  • 중요도: ⭐⭐⭐⭐⭐ (매우 자주)
  • 사용빈도: 매우 자주
  • 주요 사용처: 문자열 입력값 길이 체크
  • 암기 키워드: length = 길이
String s = "Hello";
System.out.println(s.length()); // 5

🛠️ equals(Object obj)

  • 풀네임: equals
  • 한글 뜻: 문자열 비교 (대소문자 구분)
  • 속한 클래스: String
  • 중요도: ⭐⭐⭐⭐⭐ (매우 자주)
  • 사용빈도: 매우 자주
  • 주요 사용처: 두 문자열이 같은지 비교할 때
  • 암기 키워드: equals = 같다 비교
String s1 = "Hello";
String s2 = "Hello";
System.out.println(s1.equals(s2)); // true

🛠️ equalsIgnoreCase(String anotherString)

  • 풀네임: equalsIgnoreCase
  • 한글 뜻: 문자열 비교 (대소문자 무시)
  • 속한 클래스: String
  • 중요도: ⭐⭐⭐⭐ (자주)
  • 사용빈도: 자주
  • 주요 사용처: 대소문자 무시하고 문자열 비교할 때
  • 암기 키워드: equalsIgnoreCase = 대소문자 무시 비교
String s1 = "hello";
String s2 = "HELLO";
System.out.println(s1.equalsIgnoreCase(s2)); // true

🛠️ substring(int beginIndex, int endIndex)

  • 풀네임: substring
  • 한글 뜻: 부분 문자열 추출
  • 속한 클래스: String
  • 중요도: ⭐⭐⭐⭐ (자주)
  • 사용빈도: 자주
  • 주요 사용처: 문자열의 일부분만 잘라서 사용하고 싶을 때
  • 암기 키워드: substring = 조각 추출
String s = "Hello World";
System.out.println(s.substring(0, 5)); // "Hello"

🛠️ split(String regex)

  • 풀네임: split by regular expression
  • 한글 뜻: 특정 구분자로 문자열 나누기
  • 속한 클래스: String
  • 중요도: ⭐⭐⭐⭐ (자주)
  • 사용빈도: 자주
  • 주요 사용처: 구분자를 기준으로 여러 조각으로 나누고 싶을 때
  • 암기 키워드: split = 나누기
String s = "apple,banana,orange";
String[] arr = s.split(",");
System.out.println(arr[0]); // "apple"

🛠️ replace(char oldChar, char newChar)

  • 풀네임: replace character
  • 한글 뜻: 특정 문자 교체
  • 속한 클래스: String
  • 중요도: ⭐⭐⭐⭐ (자주)
  • 사용빈도: 자주
  • 주요 사용처: 문자열 안의 특정 문자 바꾸기
  • 암기 키워드: replace = 대체
String s = "Hello";
System.out.println(s.replace('e', 'a')); // "Hallo"

🛠️ replace(CharSequence target, CharSequence replacement)

  • 풀네임: replace substring
  • 한글 뜻: 특정 문자열 교체
  • 속한 클래스: String
  • 중요도: ⭐⭐⭐⭐ (자주)
  • 사용빈도: 자주
  • 주요 사용처: 문자열 안의 특정 부분(문자열) 바꾸기
  • 암기 키워드: replace = 대체
String s = "I love Java";
System.out.println(s.replace("Java", "Python")); // "I love Python"

🛠️ toLowerCase()

  • 풀네임: to lower case
  • 한글 뜻: 모두 소문자로 변환
  • 속한 클래스: String
  • 중요도: ⭐⭐⭐⭐ (자주)
  • 사용빈도: 자주
  • 주요 사용처: 입력값 대소문자 구분 없이 처리하고 싶을 때
  • 암기 키워드: lower = 소문자
String s = "Hello";
System.out.println(s.toLowerCase()); // "hello"

  • 속한 클래스: String
  • 중요도: ⭐⭐⭐⭐ (자주)
  • 사용빈도: 자주
  • 주요 사용처: 출력 포맷을 대문자로 통일하고 싶을 때
  • 암기 키워드: upper = 대문자
String s = "hello";
System.out.println(s.toUpperCase()); // "HELLO"

🛠️ indexOf(String str)

  • 풀네임: indexOf
  • 한글 뜻: 특정 문자열의 첫 번째 위치 찾기
  • 속한 클래스: String
  • 중요도: ⭐⭐⭐ (가끔)
  • 사용빈도: 가끔
  • 주요 사용처: 문자열 내 포함 여부, 위치 찾기
  • 암기 키워드: index = 위치
String s = "Hello World";
System.out.println(s.indexOf("World")); // 6

  • 중요도: ⭐⭐⭐ (가끔)
  • 사용빈도: 가끔
  • 주요 사용처: 마지막으로 등장하는 문자열 위치 찾기
  • 암기 키워드: last = 마지막 + index = 위치
String s = "Hello Hello";
System.out.println(s.lastIndexOf("Hello")); // 6

String s = "Java Programming";

System.out.println(file.endsWith(".jpg")); // true

String s = "Hello";

🛠️ valueOf(기본형)

  • 풀네임: valueOf
  • 한글 뜻: 기본형을 문자열로 변환
  • 속한 클래스: String
  • 중요도: ⭐⭐ (가끔)
  • 사용빈도: 가끔
  • 주요 사용처: 숫자나 boolean을 문자열로 바꿔야 할 때
  • 암기 키워드: valueOf = 문자열 변환
int num = 100;
String str = String.valueOf(num);
System.out.println(str); // "100"
