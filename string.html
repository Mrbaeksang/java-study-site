📚 String 클래스 개요

Java에서 가장 많이 사용되는 클래스 중 하나인 String은 문자열을 다루는 불변(immutable) 클래스입니다.

주요 특징

  • 불변성 (Immutability)
  • String Pool을 통한 메모리 최적화
  • Thread-safe
  • Unicode 문자열 지원

🔍 기본 메서드

length()

기본
자주 사용
int length()

문자열의 길이를 반환합니다.

String str = "Hello";
int len = str.length(); // 5

charAt()

기본
자주 사용
char charAt(int index)

지정된 인덱스의 문자를 반환합니다.

String str = "Hello";
char ch = str.charAt(0); // 'H'

substring()

기본
자주 사용
String substring(int beginIndex)
String substring(int beginIndex, int endIndex)

문자열의 일부를 추출합니다.

String str = "Hello World";
String sub1 = str.substring(6); // "World"
String sub2 = str.substring(0, 5); // "Hello"

🔎 문자열 검색

indexOf()

검색
자주 사용
int indexOf(String str)
int indexOf(String str, int fromIndex)

문자열이나 문자를 앞에서부터 검색하여 위치를 반환합니다.

String str = "Hello World";
int index = str.indexOf("World"); // 6

lastIndexOf()

검색
int lastIndexOf(String str)

문자열이나 문자를 뒤에서부터 검색하여 위치를 반환합니다.

String str = "Hello Hello World";
int index = str.lastIndexOf("Hello"); // 6

🔄 문자열 변환

대소문자 변환

변환
자주 사용
String toUpperCase()
String toLowerCase()
String str = "Hello";
String upper = str.toUpperCase(); // "HELLO"
String lower = str.toLowerCase(); // "hello"

trim()

변환
자주 사용
String trim()

문자열 앞뒤의 공백을 제거합니다.

String str = " Hello ";
String trimmed = str.trim(); // "Hello"

⚖️ 문자열 비교

equals()

비교
매우 중요
boolean equals(Object obj)

두 문자열의 내용이 같은지 비교합니다.

== 연산자 대신 equals()를 사용하세요!

String str1 = "Hello";
String str2 = new String("Hello");
boolean isEqual = str1.equals(str2); // true
boolean isRef = (str1 == str2); // false

compareTo()

비교
int compareTo(String anotherString)

두 문자열을 사전순으로 비교합니다.

String str1 = "apple";
String str2 = "banana";
int result = str1.compareTo(str2); // 음수 (apple이 banana보다 앞)

📝 정규표현식

matches()

정규식
boolean matches(String regex)

문자열이 정규표현식과 일치하는지 검사합니다.

String email = "user@example.com";
boolean isValid = email.matches("^[A-Za-z0-9+_.-]+@(.+)$");

split()

정규식
자주 사용
String[] split(String regex)

정규표현식을 기준으로 문자열을 분할합니다.

String str = "apple,banana,orange";
String[] fruits = str.split(","); // ["apple", "banana", "orange"]

⚡ 성능 최적화 팁

1. String 연결 연산

반복문에서 String 연결이 필요한 경우 StringBuilder를 사용하세요.

// Bad String result = "";
for (int i = 0; i < 1000; i++) { result += i; }
// Good StringBuilder sb = new StringBuilder();
for (int i = 0; i < 1000; i++) { sb.append(i); }
String result = sb.toString();

2. String Pool 활용

리터럴로 생성된 문자열은 String Pool에 저장되어 재사용됩니다.

// String Pool 사용 String str1 = "Hello";
// 새로운 객체 생성 String str2 = new String("Hello");

💡 실전 예제

1. 이메일 유효성 검사

public static boolean isValidEmail(String email) { String regex = "^[A-Za-z0-9+_.-]+@(.+)$"; return email != null && email.matches(regex); }

2. 문자열 토큰화

String text = "Java,Python,JavaScript";
String[] languages = text.split(",");
for (String lang : languages) { System.out.println(lang.trim()); }

3. 문자열 포맷팅

String name = "John";
int age = 25;
String format = String.format("Name: %s, Age: %d", name, age);
