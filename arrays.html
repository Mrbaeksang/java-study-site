📚 Arrays API 정리

🛠️ Arrays.sort()

  • 풀네임: Arrays.sort
  • 한글 뜻: 배열 정렬
  • 속한 클래스: Arrays
  • 중요도: ⭐⭐⭐⭐⭐ (매우 자주)
  • 사용빈도: 매우 자주
  • 주요 사용처: 배열 오름차순 정렬
  • 암기 키워드: sort = 정렬
int[] arr = {5, 3, 1, 4, 2};
Arrays.sort(arr);
System.out.println(Arrays.toString(arr)); // [1, 2, 3, 4, 5]

🛠️ Arrays.copyOf()

  • 풀네임: Arrays.copyOf
  • 한글 뜻: 배열 복사
  • 속한 클래스: Arrays
  • 중요도: ⭐⭐⭐⭐ (자주)
  • 사용빈도: 자주
  • 주요 사용처: 배열 크기 확장, 복사본 만들 때
  • 암기 키워드: copy = 복사
int[] arr = {1, 2, 3};
int[] copy = Arrays.copyOf(arr, 5);
System.out.println(Arrays.toString(copy)); // [1, 2, 3, 0, 0]

🛠️ Arrays.equals()

  • 풀네임: Arrays.equals
  • 한글 뜻: 배열 내용 비교
  • 속한 클래스: Arrays
  • 중요도: ⭐⭐⭐⭐ (자주)
  • 사용빈도: 자주
  • 주요 사용처: 두 배열이 같은지 비교
  • 암기 키워드: equals = 같다
int[] arr1 = {1, 2, 3};
int[] arr2 = {1, 2, 3};
System.out.println(Arrays.equals(arr1, arr2)); // true

🛠️ Arrays.toString()

  • 풀네임: Arrays.toString
  • 한글 뜻: 배열 문자열로 변환
  • 속한 클래스: Arrays
  • 중요도: ⭐⭐⭐⭐⭐ (매우 자주)
  • 사용빈도: 매우 자주
  • 주요 사용처: 배열 출력, 디버깅
  • 암기 키워드: toString = 보기 좋게 변환
int[] arr = {1, 2, 3};
System.out.println(Arrays.toString(arr)); // [1, 2, 3]
