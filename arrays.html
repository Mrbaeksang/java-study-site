📚 Arrays 클래스 개요

java.util.Arrays는 배열을 다루는 데 필요한 다양한 메서드를 제공하는 유틸리티 클래스입니다.

주요 특징

  • 정렬 알고리즘 제공
  • 검색 기능
  • 배열 조작 유틸리티
  • 병렬 처리 지원

🔍 배열 정렬

sort()

정렬
자주 사용
sort(T[] array)
sort(T[] array, Comparator<? super T> c)

배열을 정렬합니다. 기본형은 오름차순, 객체는 Comparable 구현 필요

// 기본형 배열 정렬 int[] numbers = {5, 2, 8, 1, 9};
Arrays.sort(numbers); // [1, 2, 5, 8, 9]
// 객체 배열 정렬 String[] words = {"banana", "apple", "cherry"};
Arrays.sort(words); // ["apple", "banana", "cherry"]

parallelSort()

정렬
병렬처리
parallelSort(T[] array)

멀티스레드를 사용하여 배열을 병렬로 정렬합니다.

int[] bigArray = new int[1000000];
// 큰 배열의 경우 병렬 정렬이 더 빠를 수 있음 Arrays.parallelSort(bigArray);

🔎 배열 검색

binarySearch()

검색
자주 사용
binarySearch(T[] array, T key)

배열은 반드시 정렬되어 있어야 합니다!

int[] numbers = {1, 2, 5, 8, 9};
int index = Arrays.binarySearch(numbers, 5); // 2

📋 배열 복사

copyOf()

복사
자주 사용
copyOf(T[] original, int newLength)

배열을 새로운 길이로 복사합니다.

int[] original = {1, 2, 3};
int[] copy = Arrays.copyOf(original, 5); // [1, 2, 3, 0, 0]

copyOfRange()

복사
copyOfRange(T[] original, int from, int to)

배열의 특정 범위를 복사합니다.

int[] original = {1, 2, 3, 4, 5};
int[] copy = Arrays.copyOfRange(original, 1, 4); // [2, 3, 4]

⚖️ 배열 비교

equals()

비교
자주 사용
equals(Object[] a, Object[] b)

두 배열의 내용이 같은지 비교합니다.

int[] arr1 = {1, 2, 3};
int[] arr2 = {1, 2, 3};
boolean isEqual = Arrays.equals(arr1, arr2); // true

deepEquals()

비교
deepEquals(Object[] a1, Object[] a2)

다차원 배열의 내용을 재귀적으로 비교합니다.

int[][] arr1 = {{1, 2}, {3, 4}};
int[][] arr2 = {{1, 2}, {3, 4}};
boolean isEqual = Arrays.deepEquals(arr1, arr2); // true

🎨 배열 채우기

fill()

채우기
fill(T[] array, T value)

배열의 모든 요소를 지정된 값으로 채웁니다.

int[] array = new int[5];
Arrays.fill(array, 10); // [10, 10, 10, 10, 10]

🌊 스트림 변환

stream()

스트림
Java 8+
stream(T[] array)

배열을 스트림으로 변환합니다.

String[] words = {"hello", "world"};
Arrays.stream(words) .map(String::toUpperCase) .forEach(System.out::println);

⚡ 성능 최적화 팁

1. 정렬 알고리즘 선택

대용량 데이터의 경우 parallelSort()를 사용하면 성능이 향상될 수 있습니다.

// 작은 배열 Arrays.sort(smallArray);
// 큰 배열 Arrays.parallelSort(bigArray);

2. 검색 최적화

binarySearch()는 정렬된 배열에서만 사용하세요.

// 정렬되지 않은 배열에서 검색 - 선형 시간 for (int i = 0; i < arr.length; i++) { if (arr[i] == target) return i; }
// 정렬된 배열에서 검색 - 로그 시간 Arrays.sort(arr); int index = Arrays.binarySearch(arr, target);

💡 실전 예제

1. 커스텀 객체 정렬

class Person { String name; int age; // 생성자, getter, setter }
Person[] people = {new Person("John", 25), new Person("Alice", 20)};
// 나이순 정렬 Arrays.sort(people, (p1, p2) -> p1.age - p2.age);

2. 2차원 배열 처리

int[][] matrix = {{1, 2}, {3, 4}};
// 깊은 복사 int[][] copy = new int[matrix.length][];
for (int i = 0; i < matrix.length; i++) { copy[i] = Arrays.copyOf(matrix[i], matrix[i].length); }

3. 배열을 리스트로 변환

String[] array = {"apple", "banana", "cherry"};
List list = Arrays.asList(array);
// 또는 Java 8+ List list2 = Arrays.stream(array).collect(Collectors.toList());
