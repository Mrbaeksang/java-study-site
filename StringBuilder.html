🧵 StringBuilder / StringBuffer 정리

🛠️ StringBuilder.append()

  • 풀네임: StringBuilder.append
  • 한글 뜻: 문자열 끝에 추가
  • 속한 클래스: StringBuilder
  • 중요도: ⭐⭐⭐⭐⭐ (매우 자주)
  • 사용빈도: 매우 자주
  • 주요 사용처: 반복 문자열 추가 시 (예: 반복문 안에서)
  • 암기 키워드: append = 뒤에 붙이기
StringBuilder sb = new StringBuilder();
sb.append("Hello");
sb.append(" World");
System.out.println(sb.toString()); // Hello World

🛠️ StringBuilder.insert()

  • 풀네임: StringBuilder.insert
  • 한글 뜻: 특정 위치에 문자열 삽입
  • 속한 클래스: StringBuilder
  • 중요도: ⭐⭐⭐ (가끔)
  • 사용빈도: 가끔
  • 주요 사용처: 중간에 문자열 추가할 때
  • 암기 키워드: insert = 삽입
StringBuilder sb = new StringBuilder("Hello");
sb.insert(5, " World");
System.out.println(sb.toString()); // Hello World

🛠️ StringBuilder.delete()

  • 풀네임: StringBuilder.delete
  • 한글 뜻: 문자열 일부 삭제
  • 속한 클래스: StringBuilder
  • 중요도: ⭐⭐⭐ (가끔)
  • 사용빈도: 가끔
  • 주요 사용처: 필요 없는 문자열 제거할 때
  • 암기 키워드: delete = 삭제
StringBuilder sb = new StringBuilder("Hello World");
sb.delete(5, 11);
System.out.println(sb.toString()); // Hello

🛠️ StringBuilder.replace()

  • 풀네임: StringBuilder.replace
  • 한글 뜻: 문자열 일부 교체
  • 속한 클래스: StringBuilder
  • 중요도: ⭐⭐⭐ (가끔)
  • 사용빈도: 가끔
  • 주요 사용처: 특정 부분 문자열 교체할 때
  • 암기 키워드: replace = 교체
StringBuilder sb = new StringBuilder("Hello World");
sb.replace(6, 11, "Java");
System.out.println(sb.toString()); // Hello Java

🛠️ StringBuilder.reverse()

  • 풀네임: StringBuilder.reverse
  • 한글 뜻: 문자열 뒤집기
  • 속한 클래스: StringBuilder
  • 중요도: ⭐⭐ (드물게)
  • 사용빈도: 드물게
  • 주요 사용처: 문자열을 뒤집을 때
  • 암기 키워드: reverse = 뒤집기
StringBuilder sb = new StringBuilder("Hello");
sb.reverse();
System.out.println(sb.toString()); // olleH

🛠️ StringBuilder.toString()

  • 풀네임: StringBuilder.toString
  • 한글 뜻: StringBuilder를 문자열(String)로 변환
  • 속한 클래스: StringBuilder
  • 중요도: ⭐⭐⭐⭐⭐ (매우 자주)
  • 사용빈도: 매우 자주
  • 주요 사용처: 최종 결과물을 String 타입으로 얻을 때
  • 암기 키워드: toString = 문자열로 변환
StringBuilder sb = new StringBuilder();
sb.append("Hello");
String result = sb.toString();
System.out.println(result); // Hello
