🧮 Math API 정리

🛠️ Math.abs()

  • 풀네임: Math.abs
  • 한글 뜻: 절댓값 반환
  • 속한 클래스: Math
  • 중요도: ⭐⭐⭐⭐⭐ (매우 자주)
  • 사용빈도: 매우 자주
  • 주요 사용처: 값의 부호에 관계없이 거리, 차이 계산할 때
  • 암기 키워드: abs = 절댓값
int a = Math.abs(-10);
System.out.println(a); // 10

🛠️ Math.max()

  • 풀네임: Math.max
  • 한글 뜻: 두 수 중 큰 값 반환
  • 속한 클래스: Math
  • 중요도: ⭐⭐⭐⭐⭐ (매우 자주)
  • 사용빈도: 매우 자주
  • 주요 사용처: 최댓값 찾기, 조건 비교
  • 암기 키워드: max = 최대
int a = Math.max(7, 3);
System.out.println(a); // 7

🛠️ Math.min()

  • 풀네임: Math.min
  • 한글 뜻: 두 수 중 작은 값 반환
  • 속한 클래스: Math
  • 중요도: ⭐⭐⭐⭐⭐ (매우 자주)
  • 사용빈도: 매우 자주
  • 주요 사용처: 최솟값 찾기, 범위 설정
  • 암기 키워드: min = 최소
int a = Math.min(7, 3);
System.out.println(a); // 3

🛠️ Math.pow()

  • 풀네임: Math.pow
  • 한글 뜻: 거듭제곱 계산
  • 속한 클래스: Math
  • 중요도: ⭐⭐⭐⭐ (자주)
  • 사용빈도: 자주
  • 주요 사용처: 제곱 계산, 지수 연산
  • 암기 키워드: pow = power (거듭제곱)
double a = Math.pow(2, 3);
System.out.println(a); // 8.0

🛠️ Math.sqrt()

  • 풀네임: Math.sqrt
  • 한글 뜻: 제곱근 반환
  • 속한 클래스: Math
  • 중요도: ⭐⭐⭐ (가끔)
  • 사용빈도: 가끔
  • 주요 사용처: 피타고라스 정리, 거리 계산
  • 암기 키워드: sqrt = square root (제곱근)
double a = Math.sqrt(9);
System.out.println(a); // 3.0

🛠️ Math.random()

  • 풀네임: Math.random
  • 한글 뜻: 0 이상 1 미만 난수 생성
  • 속한 클래스: Math
  • 중요도: ⭐⭐⭐⭐⭐ (매우 자주)
  • 사용빈도: 매우 자주
  • 주요 사용처: 랜덤 값 생성 (예: 추첨, 시뮬레이션)
  • 암기 키워드: random = 랜덤
double r = Math.random();
System.out.println(r); // 0.0 이상 1.0 미만 값

System.out.println(r); // 3.0
