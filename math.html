📚 Math 클래스 개요

java.lang.Math는 수학적 연산을 위한 정적 메서드들을 제공하는 유틸리티 클래스입니다.

주요 특징

  • 모든 메서드가 static
  • 기본적인 수학 연산 제공
  • 삼각함수 지원
  • 지수/로그 함수 지원

🔍 기본 연산

abs()

기본
자주 사용
abs(double a)

숫자의 절대값을 반환합니다.

double num = -5.7;
double abs = Math.abs(num); // 5.7

max() / min()

기본
자주 사용
max(double a, double b)
min(double a, double b)
double max = Math.max(10.5, 20.7); // 20.7
double min = Math.min(10.5, 20.7); // 10.5

pow()

지수
자주 사용
pow(double base, double exponent)

거듭제곱을 계산합니다.

double result = Math.pow(2, 3); // 8.0 (2³)

🎯 반올림/올림/내림

round()

반올림
자주 사용
round(double a)

가장 가까운 정수로 반올림합니다.

double num = 3.7;
long rounded = Math.round(num); // 4

ceil()

올림
ceil(double a)

주어진 숫자보다 크거나 같은 가장 작은 정수를 반환합니다.

double num = 3.1;
double ceiling = Math.ceil(num); // 4.0

floor()

내림
floor(double a)

주어진 숫자보다 작거나 같은 가장 큰 정수를 반환합니다.

double num = 3.9;
double floored = Math.floor(num); // 3.0

📐 삼각함수

sin()

삼각함수
sin(double theta)

각도의 사인값을 반환합니다 (라디안 단위).

double angle = Math.PI / 2; // 90도
double result = Math.sin(angle); // 1.0

cos()

삼각함수
cos(double theta)

각도의 코사인값을 반환합니다 (라디안 단위).

double angle = Math.PI; // 180도
double result = Math.cos(angle); // -1.0

tan()

삼각함수
tan(double theta)

각도의 탄젠트값을 반환합니다 (라디안 단위).

double angle = Math.PI / 4; // 45도
double result = Math.tan(angle); // 1.0

각도 변환

// 도(degree)를 라디안(radian)으로 변환 double radian = degree * Math.PI / 180.0;
// 라디안을 도로 변환 double degree = radian * 180.0 / Math.PI;

📈 로그/지수 함수

log()

로그
log(double a)

자연로그(밑이 e인 로그)를 계산합니다.

double result = Math.log(Math.E); // 1.0

exp()

지수
exp(double a)

e의 거듭제곱을 계산합니다.

double result = Math.exp(1); // 2.718... (e¹)

💡 실전 예제

1. 원의 넓이 계산

public static double calculateCircleArea(double radius) { return Math.PI * Math.pow(radius, 2); }

2. 두 점 사이의 거리 계산

public static double calculateDistance(double x1, double y1, double x2, double y2) { double dx = x2 - x1; double dy = y2 - y1; return Math.sqrt(Math.pow(dx, 2) + Math.pow(dy, 2)); }

3. 숫자 반올림 (소수점 지정)

public static double round(double value, int places) { double scale = Math.pow(10, places); return Math.round(value * scale) / scale; }

⚡ 성능 최적화 팁

1. 상수 활용

자주 사용하는 값은 Math 클래스의 상수를 활용하세요.

// Bad double pi = 3.14159;
// Good double pi = Math.PI;

2. BigDecimal 사용

정확한 소수점 계산이 필요한 경우 BigDecimal을 사용하세요.

// 부정확할 수 있음 double result = 0.1 + 0.2;
// 정확한 계산 BigDecimal a = new BigDecimal("0.1"); BigDecimal b = new BigDecimal("0.2"); BigDecimal result = a.add(b);
