2️⃣ CREATE, INSERT, SELECT

테이블 생성과 기본 데이터 조회를 배워보세요.

1. CREATE TABLE 문

CREATE TABLE 문은 새로운 테이블을 생성하는 SQL 명령어입니다.

📝 예제: 학생 정보 테이블 생성

CREATE TABLE students ( student_id INT PRIMARY KEY, name VARCHAR(50) NOT NULL, department VARCHAR(50), grade INT, admission_date DATE );

2. INSERT INTO 문

INSERT INTO 문은 테이블에 새로운 데이터를 삽입합니다.

📝 예제: 학생 데이터 삽입

INSERT INTO students (student_id, name, department, grade, admission_date) VALUES (20230001, '김철수', '컴퓨터공학', 2, '2023-03-01');

3. SELECT 문 기본

SELECT 문은 테이블에서 데이터를 조회하는 가장 기본적인 SQL 명령어입니다.

📝 예제: 모든 열 조회

SELECT * FROM students;

📝 예제: 특정 열만 조회

SELECT name, department FROM students;

💡 팁: 조회 성능 최적화

필요한 열만 SELECT 문에 포함시키는 것이 성능에 좋습니다.

4. 데이터 타입

데이터 타입 설명 예시
INT 정수형 20230001
VARCHAR(n) 문자열 (최대 n자) '김철수'
DATE 날짜형 '2023-03-01'
