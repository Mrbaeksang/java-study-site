관련 메서드 상세 설명

nextInt(int bound)

// 예시 Random random = new Random(); int number = random.nextInt(100);
  • 사용처: 범위 내 난수 생성
  • 성능: O(1)
  • 실전 활용: 게임, 테스트 데이터 생성
  • 주의사항:
    • bound 범위 주의
    • 성능 고려
    • 시드값 고려

nextDouble()

// 예시 Random random = new Random(); double number = random.nextDouble();
  • 사용처: 실수 난수 생성
  • 성능: O(1)
  • 실전 활용: 확률 계산, 시뮬레이션
  • 주의사항:
    • 정밀도 주의
    • 성능 고려
    • 시드값 고려
