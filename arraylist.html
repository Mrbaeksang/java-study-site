📚 ArrayList API 정리

🛠️ add()

  • 풀네임: ArrayList.add
  • 한글 뜻: 리스트에 요소 추가
  • 속한 클래스: ArrayList
  • 중요도: ⭐⭐⭐⭐⭐ (매우 자주)
  • 사용빈도: 매우 자주
  • 주요 사용처: 새로운 데이터 추가할 때
  • 암기 키워드: add = 추가
ArrayList list = new ArrayList<>();
list.add(10);
list.add(20);
System.out.println(list); // [10, 20]

🛠️ get()

  • 풀네임: ArrayList.get
  • 한글 뜻: 특정 인덱스 요소 가져오기
  • 속한 클래스: ArrayList
  • 중요도: ⭐⭐⭐⭐⭐ (매우 자주)
  • 사용빈도: 매우 자주
  • 주요 사용처: 저장된 데이터 읽을 때
  • 암기 키워드: get = 꺼내기
System.out.println(list.get(0)); // 10
