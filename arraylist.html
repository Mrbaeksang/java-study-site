📚 ArrayList 개요

ArrayList는 크기가 동적으로 조절되는 배열 기반의 List 구현체입니다.

주요 특징

  • 동적 크기 조절
  • 순차적 데이터 접근
  • 인덱스 기반 검색
  • 제네릭 지원

Array vs ArrayList

특성 Array ArrayList
크기 고정 동적
타입 기본형/참조형 참조형만
성능 빠름 약간 느림
유연성 낮음 높음

🛠 기본 메서드

add()

추가
자주 사용
add(E element)
add(int index, E element)

요소를 추가합니다.

ArrayList list = new ArrayList<>(); list.add("Apple"); // 끝에 추가 list.add(0, "Banana"); // 인덱스 0에 추가

remove()

삭제
remove(Object o)
remove(int index)

요소를 제거합니다.

list.remove("Apple"); // 객체로 제거 list.remove(0); // 인덱스로 제거

get()

조회
자주 사용
get(int index)

특정 위치의 요소를 반환합니다.

String item = list.get(0); // 첫 번째 요소 가져오기

🔍 검색과 수정

contains()

검색
contains(Object o)

요소의 포함 여부를 확인합니다.

boolean exists = list.contains("Apple"); // true/false

indexOf()

검색
indexOf(Object o)

요소의 위치를 찾습니다.

int index = list.indexOf("Apple"); // 위치 또는 -1

set()

수정
set(int index, E element)

특정 위치의 요소를 교체합니다.

list.set(0, "Orange"); // 첫 번째 요소 교체

💡 실전 예제

1. 중복 제거

public static ArrayList removeDuplicates(ArrayList list) { return new ArrayList<>(new LinkedHashSet<>(list)); } // 사용 예 ArrayList fruits = new ArrayList<>( Arrays.asList("Apple", "Banana", "Apple", "Orange") ); fruits = removeDuplicates(fruits); // [Apple, Banana, Orange]

2. 안전한 요소 제거

public static void removeIf(ArrayList list, Predicate condition) { for (int i = list.size() - 1; i >= 0; i--) { if (condition.test(list.get(i))) { list.remove(i); } } } // 사용 예 ArrayList numbers = new ArrayList<>( Arrays.asList(1, 2, 3, 4, 5) ); removeIf(numbers, n -> n % 2 == 0); // 짝수 제거

3. 리스트 분할

public static ArrayList> partition(ArrayList list, int size) { ArrayList> parts = new ArrayList<>(); for (int i = 0; i < list.size(); i += size) { parts.add(new ArrayList<>( list.subList(i, Math.min(list.size(), i + size)) )); } return parts; } // 사용 예 ArrayList numbers = new ArrayList<>( Arrays.asList(1, 2, 3, 4, 5, 6) ); ArrayList> parts = partition(numbers, 2); // [[1, 2], [3, 4], [5, 6]]

💡 실전 예제

1. 람다식을 사용한 정렬

// Student 클래스 정의 class Student { String name; int age; double score; public Student(String name, int age, double score) { this.name = name; this.age = age; this.score = score; } } // 학생 리스트 생성 ArrayList students = new ArrayList<>(); students.add(new Student("김철수", 20, 85.5)); students.add(new Student("이영희", 19, 92.0)); students.add(new Student("박민수", 21, 88.5)); // 나이순 정렬 (오름차순) students.sort((s1, s2) -> s1.age - s2.age); // 이름순 정렬 students.sort((s1, s2) -> s1.name.compareTo(s2.name)); // 성적순 정렬 (내림차순) students.sort((s1, s2) -> Double.compare(s2.score, s1.score)); // Comparator 체이닝을 사용한 다중 조건 정렬 students.sort( Comparator.comparing(Student::getAge) // 먼저 나이로 정렬 .thenComparing(Student::getName) // 나이가 같으면 이름으로 정렬 .thenComparingDouble(Student::getScore) // 나이와 이름이 같으면 성적으로 정렬 );

실무 활용 팁

  • 단순 비교는 s1.age - s2.age 형태로 작성 가능
  • 문자열 비교는 compareTo() 메서드 사용
  • 실수 비교는 Double.compare() 사용 권장
  • 여러 조건으로 정렬할 때는 Comparator 체이닝 활용

2. 필터링과 정렬 조합

// 성적이 85점 이상인 학생들만 선택해서 나이순으로 정렬 ArrayList highScoreStudents = new ArrayList<>(); for (Student s : students) { if (s.score >= 85.0) { highScoreStudents.add(s); } } highScoreStudents.sort((s1, s2) -> s1.age - s2.age); // Stream API를 사용한 더 간단한 방법 List result = students.stream() .filter(s -> s.score >= 85.0) // 성적 필터링 .sorted((s1, s2) -> s1.age - s2.age) // 나이순 정렬 .collect(Collectors.toList());

Stream API vs 전통적인 방법

  • Stream API: 코드가 간결하고 가독성이 좋음
  • 전통적인 방법: 성능이 중요한 경우 선호됨
  • 대용량 데이터: 상황에 따라 적절한 방법 선택

3. 실무에서 자주 사용되는 패턴

// 특정 조건을 만족하는 요소 찾기 Student topStudent = students.stream() .max((s1, s2) -> Double.compare(s1.score, s2.score)) .orElse(null); // 평균 계산 double averageScore = students.stream() .mapToDouble(s -> s.score) .average() .orElse(0.0); // 그룹화 Map> studentsByAge = students.stream() .collect(Collectors.groupingBy(s -> s.age)); // 커스텀 정렬 기준 정의 class StudentComparator implements Comparator { @Override public int compare(Student s1, Student s2) { // 먼저 성적으로 내림차순 정렬 int scoreCompare = Double.compare(s2.score, s1.score); if (scoreCompare != 0) { return scoreCompare; } // 성적이 같으면 나이로 오름차순 정렬 int ageCompare = s1.age - s2.age; if (ageCompare != 0) { return ageCompare; } // 나이도 같으면 이름으로 정렬 return s1.name.compareTo(s2.name); } } // 커스텀 정렬 사용 students.sort(new StudentComparator());
